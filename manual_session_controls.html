BD Creator - Contrôles de session manuels

Instructions pour ajouter manuellement les contrôles de session

Si le gestionnaire de sessions ne parvient pas à ajouter automatiquement les contrôles, vous pouvez les ajouter manuellement en suivant ces étapes :

1. Copiez le bloc HTML suivant :

<div class="session-controls"> <div class="session-buttons"> <button id="save-session-btn" class="button session-button">Sauvegarder la session</button> <button id="new-session-btn" class="button session-button">Nouvelle session</button> <select id="load-session-select" class="session-select"> <option value="">Charger une session...</option> </select> </div> <div id="session-info" class="session-info"> Session actuelle: <span id="current-session-name">Nouvelle session</span> </div> </div>

2. Copiez les styles CSS suivants :

<style> .session-controls { background-color: #f0f0f0; padding: 10px; border-radius: 5px; margin-bottom: 20px; display: flex; justify-content: space-between; align-items: center; flex-wrap: wrap; } .session-buttons { display: flex; gap: 10px; flex-wrap: wrap; } .session-button { background-color: #2980b9; color: white; padding: 10px 20px; text-decoration: none; border-radius: 5px; font-weight: bold; margin-right: 10px; cursor: pointer; border: none; } .session-select { padding: 10px; border-radius: 5px; border: 1px solid #ddd; background-color: white; min-width: 200px; } .session-info { margin-top: 10px; font-style: italic; color: #555; } #new-session-btn { background-color: #27ae60; } #delete-session-btn { background-color: #e74c3c; } </style>

3. Insérez le bloc HTML dans chaque fichier HTML :

Placez le bloc HTML juste après l'ouverture de votre div principal avec la classe "container" :

<div class="container"> <h1>BD Creator - [Titre de la page]</h1> <div class="session-controls"> ... </div> <div class="navigation"> ... </div> </div>

4. Insérez les styles CSS dans chaque fichier HTML :

Placez les styles CSS dans la section head de chaque fichier HTML :

<head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>BD Creator</title> <style> /* Styles existants */ </style> <style> .session-controls { ... } ... </style> </head>

5. Assurez-vous que le script du gestionnaire de sessions est bien inclus :

Vérifiez que cette ligne est présente à la fin de chaque fichier HTML, juste avant la fermeture de la balise body :

<script src="js/bd_creator_session_manager.js"></script>

Si vos fichiers HTML sont dans le dossier js, utilisez :

<script src="bd_creator_session_manager.js"></script>

Si vos fichiers HTML sont dans le dossier css, utilisez :

<script src="../js/bd_creator_session_manager.js"></script>

Débogage du gestionnaire de sessions

Si vous rencontrez toujours des problèmes après avoir ajouté manuellement les contrôles, essayez d'utiliser la version de débogage du gestionnaire de sessions :

  1. Remplacez votre fichier bd_creator_session_manager.js par la version de débogage fournie
  2. Ouvrez la console du navigateur (F12) pour voir les messages de débogage
  3. Vérifiez s'il y a des erreurs ou des avertissements
